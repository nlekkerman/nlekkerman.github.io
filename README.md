## Projects

### [HotelsMates](https://github.com/nlekkerman/HotelMateFrontend) — Featured Project

Multi-tenant hospitality platform designed for real operations. Staff onboarding, booking lifecycle, guest communication, live dashboards.

- **Multi-Tenant Isolation** — Each hotel in its own scoped environment
- **Live Operational State** — Event-driven dashboards via Pusher
- **Role-Based Access** — Granular permissions (owner, manager, receptionist, housekeeper)
- **Full Booking Lifecycle** — Reservation → Pre-check-in → In-house → Checkout → Post-stay

**Tech**: Python, Django, React, PostgreSQL, Pusher, REST APIs

[Frontend Code](https://github.com/nlekkerman/HotelMateFrontend) · [Backend Code](https://github.com/nlekkerman/HotelMateBackend)
